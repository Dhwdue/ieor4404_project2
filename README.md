#  IEOR 4404 Project 2 — NBA Schedule Feasibility and Optimization

This repository contains the code and data used to analyze the draft NBA schedule and formulate integer programming models for feasibility and travel minimization.

## Contents
- **Final_Code.ipynb** — Jupyter notebook with all four parts of the project.
- **games.csv** — Draft schedule provided for analysis.
- **schedule_part4.csv** — Optimized schedule minimizing total time zone travel.
- **LICENSE** — MIT license.

## Reproducibility
All models run using Python, Gurobi, and the data included in this repository.
